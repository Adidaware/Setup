# VM Setup (Linux/Debian)

## Prerequisites
- Linux VM (Debian-based)
- bash shell
- sudo access

## What's included
- Aliases for quick folder access
- Functions for cleanup and utilities
- Custom .bashrc configuration

## How to apply these settings:

1. Copy aliases from `aliases.md` to your `~/.bashrc`
2. Copy functions from `functions.md` to your `~/.bashrc`
3. Run: `source ~/.bashrc`

## Files
- `aliases.md` - All your VM aliases
- `functions.md` - All your VM functions
- `dotfiles/.bashrc` - Full .bashrc file (reference)
