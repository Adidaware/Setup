# VM Aliases

## Quick Folder Access

### `outputs` - Go to SVABA outputs folder
**What it does:** Quick cd to outputs directory

**Command:**
```bash
alias outputs='cd /mnt/wgs2/svaba_outputs'
```

**How to add:**
1. Open `~/.bashrc`
2. Add the line above
3. Run: `source ~/.bashrc`
4. Test: `outputs`

---

### `cdwgs` - Go to WGS data folder
**What it does:** Quick access to WGS data directory

**Command:**
```bash
alias cdwgs='cd /mnt/wgs2'
```

---

### `cdhome` - Go to home directory
**What it does:** Quick return to home folder

**Command:**
```bash
alias cdhome='cd ~'
```

---

### `size` - List files with human-readable sizes
**What it does:** Shortcut for `ls -lh` to see file sizes in KB/MB/GB

**Command:**
```bash
alias size='ls -lh'
```

---

### `wgs` - Go to WGS data folder
**What it does:** Quick cd to primary WGS data directory

**Command:**
```bash
alias wgs='cd /mnt/wgs'
```

---

### `wgs2` - Go to WGS2 data folder
**What it does:** Quick cd to secondary WGS data directory

**Command:**
```bash
alias wgs2='cd /mnt/wgs2'
```

---

### `scripts` - Go to svaba scripts folder
**What it does:** Quick cd to the svaba build scripts directory

**Command:**
```bash
alias scripts='cd /home/aditya/svaba/build/svaba-working/scripts'
```

**How to add:**
1. Open `~/.bashrc`
2. Add the line above
3. Run: `source ~/.bashrc`
4. Test: `scripts`
