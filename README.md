# My Development Setup Guide

This repo contains all my configuration for:
- **Mac (Local)** - zsh shell, GCloud CLI, aliases for VM access
- **VM (Linux)** - bash shell, data processing tools, quick access aliases

## Quick Start

### On a new Mac:
```bash
cd mac/
bash setup-mac.sh  # (if you create this later)
```

### On a new VM:
```bash
cd vm/
bash setup-vm.sh  # (if you create this later)
```

## Directory Structure
- `mac/` - All Mac-specific setup
- `vm/` - All VM-specific setup
- `setup-scripts/` - Automated setup scripts (optional)
