# Mac Setup

## Prerequisites
- macOS with zsh (Catalina or later)
- Homebrew installed
- Google Cloud SDK (`gcloud` command)

## What's included
- Aliases for VM access
- Functions for downloading files from VM
- Custom .zprofile configuration

## How to apply these settings:

1. Copy aliases from `aliases.md` to your `~/.zprofile`
2. Copy functions from `functions.md` to your `~/.zprofile`
3. Run: `source ~/.zprofile`

## Files
- `aliases.md` - All your Mac aliases
- `functions.md` - All your Mac functions
- `dotfiles/.zprofile` - Full .zprofile file (reference)
